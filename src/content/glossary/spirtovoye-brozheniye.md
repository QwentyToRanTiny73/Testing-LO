Спиртовое брожение — центральный процесс виноделия: гексозы (глюкоза, фруктоза) через гликолиз (путь Эмбдена–Мейергофа–Парнаса) превращаются в пируват, который затем декарбоксилируется до ацетальдегида и восстанавливается до этанола. Суммарно: С₆Н₁₂О₆ → 2 С₂Н₅ОН + 2 СО₂ + тепло.

**Практическое уравнение.** Ориентировочно 1 °Brix (1 г/100 мл сахара) даёт ≈ 0,56 об.% спирта; 1 г/л сахара → ≈ 0,06 об.% (оба значения — ориентиры для расчётов, зависят от штамма и условий).

**Побочные продукты.** Помимо этанола и CO₂, образуются: глицерол (≈3–10% массы сброженных сахаров), сивушные масла (высшие спирты), эфиры, органические кислоты (янтарная, молочная), ацетальдегид, SO₂ (в малых количествах дрожжами).

**Стадии.**
1. *Лаг-фаза* — адаптация дрожжей, рост популяции (ориентировочно 1–3 сут).
2. *Турбулентное (активное) брожение* — интенсивное выделение CO₂, быстрое потребление сахаров.
3. *Тихое брожение* — замедление при снижении сахаров, критична управляемость температурой.
4. *Доброжевание* — вывод на сухость (остаточные сахара < 2 г/л для сухих вин).

**Ключевые параметры.** Температура (ориентировочно 12–16 °C для ароматических белых, 22–28 °C для красных с мацерацией), обеспеченность ЯСА, начальная концентрация сахара, pH, популяция и штамм дрожжей.
